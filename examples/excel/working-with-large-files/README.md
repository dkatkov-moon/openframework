# Working with large files

This process example demonstrates Excel package functionality can work with large Excel files.  

## Configuration
All necessary configuration files can be found in <code>src/main/resources</code> directory.

**apm_run.properties**

| Parameter     | Value         |
| ------------- |---------------|
| `source.spreadsheet.file` | Path to the source spreadsheet file. It can be path on local file system or within resources of this project. |

## Running

Run `main()` method of `LocalRunner` class.
